## ūüöÄ Running Locally

### 1. Clone the repository

```bash
git clone https://github.com/your-username/hospital-management-system.git
cd hospital-management-system
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Configure your database

Create a `.env` file or set environment variables:

```env
DB_HOST=your_mysql_host
DB_USER=your_mysql_user
DB_PASSWORD=your_mysql_password
DB_NAME=your_database_name
```

### 4. Run the app

```bash
python app.py
```

Visit `http://localhost:5000` in your browser.

## ūüóĄÔłŹ Database Schema

```sql
patients       ‚Üí patient_id, name, age, gender, phone
doctors        ‚Üí doctor_id, name, specialization, phone, email
appointments   ‚Üí appointment_id, patient_id (FK), doctor_id (FK), date, time
treatment_records ‚Üí record_id, appointment_id (FK), notes, medication
```

> Cascaded deletes are handled at the **application layer** ‚ÄĒ deleting a doctor or patient also removes their appointments and treatment records in the correct order to avoid integrity errors.
